# Touch Designer | Windows Manager 🌌

A premium, hand-tracked window management environment focused on neural aesthetics and immersive gesture-driven interactions.

## 🚀 Key Features

### Gesture-Driven Navigation
- **Create Windows**: Pinch both hands together and stretch to spawn glassmorphic frames.
- **Move Windows**: Left-hand pinch (**2s hold**) to pick up and reposition frames.
- **Delete Windows**: Left-hand fist (**5s hold**) to close frames instantly—no confirmation required.
- **Quick Actions**: Right-hand index finger acts as a virtual cursor with a quick pinch-to-click gesture.

### Contextual Content Menu
- **Activate**: Right-hand pinch (**1s hold**) over any frame.
- **Interact**: Move hand up for **Google** or down for **YouTube**.
- **Unified Feedback**: A cyan neural progress ring visualizes all hold-based triggers.

## 🛠️ Setup & Requirements

- **Browser**: Best experienced in Chrome or Edge with a high-quality webcam.
- **Server Required**: To avoid YouTube/Google iframe security restrictions (Error 153), this project must be served via HTTP/S.
  - **Local**: Run `npx serve .` or use VS Code Live Server.
  - **Cloud**: Deploy directly to Vercel or Netlify.
- **Environment**: Ensure a well-lit space for optimal MediaPipe hand tracking performance.
